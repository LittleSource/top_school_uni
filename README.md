# TOP校园

uni-app写的第一版TOP校园

此项目完工于2018.12，历时半年，本以为它会成为一个家喻户晓的软件，却没想到在淹没在一个不为人知的角落。

那么为什么不把它开源出来让需要的人有所帮助呢？

这个项目是我第一次写的非常庞大的项目，虽然各个方面都体现出了菜鸟的手法，但是在学习技术这一方面来说对我有着深远的影响。

敬以此纪念我半年的时光和我曾热爱的梦想~

------



### 主要功能：

- 三方登陆支付

- 文章发布
- 评论回复
- websocket即时消息
- 商城相关
- 兼职内容发布展示

- ...
